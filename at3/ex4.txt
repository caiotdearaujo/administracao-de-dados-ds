4. Defina o que é um sistema gerenciador de banco de dados (SGDB)? Dê exemplos de banco de dados existentes.

O SGBD é um software responsável pelo gerenciamento de um ou mais bancos de dados. Seu trabalho é gerenciar o acesso, a manipulação e a organização dos bancos de dados, por exemplo. SGBD's de bancos de dados relacionais (SQL) requerem a inserção de dados com seus tipos e relações, negando a inserção fora dos padrões. Em suma, o SGBD relacional disponibiliza uma interface para que seus clientes possam incluir, alterar ou consultar dados previamente armazenados. Exemplos: PostgreSQL, MySQL, Oracle, SQL Server, MariaDB, entre outros.
